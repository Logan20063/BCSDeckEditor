# Bloons Card Storm Deck Editor

## A browser-based deck editor for Bloons Card Storm that allows users to create decks and export them as text files or images.

**Website:** https://logan20063.github.io/BCSDeckEditor/

## Features:
- Card Searching
- Card Sorting
- Game-like UI
- Importing and Exporting deck to .txt files
- Downloading deck as Images

## Technologies Used
- HTML
- CSS
- JS
- Github Pages
